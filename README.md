# Demo-Java-test

## Project Summary:
This project has demo test using Java

## Technology used:
- IntelliJ IDEA

## Pre requisites:
- Apache JMETER

## How to run
- ``` git clone ```
- ``` npm i ```

A report will be generate in your project source root
